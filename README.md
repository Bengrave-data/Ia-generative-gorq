# 🤖 IA Générative avec Groq et Llama 3.3

Projet de développement d'applications d'intelligence artificielle générative utilisant l'API Groq et le modèle Llama 3.3 de Meta.

## 🎯 Objectif

Construire des applications IA concrètes et utilisables en business : chatbots, générateurs de contenu, assistants personnalisés. Démontrer la maîtrise des modèles de langage (LLM) de dernière génération.

## 🛠️ Technologies utilisées

- **Python 3**
- **Groq API** — inférence ultra-rapide de LLM
- **Llama 3.3 (70 milliards de paramètres)** — modèle de Meta
- **Google Colab Secrets** — gestion sécurisée des clés API

## 💡 Fonctionnalités développées

### 1. Appels API de base
Premier contact avec un LLM via une API standardisée (compatible OpenAI).

### 2. Assistant spécialisé (générateur de posts LinkedIn)
- Utilisation de **system prompts** pour spécialiser l'IA
- Paramétrage de la **température** pour contrôler la créativité
- Génération de contenu marketing structuré

### 3. Chatbot conversationnel avec mémoire
- Gestion d'un **historique de conversation**
- Maintien du contexte entre les messages
- Comportement similaire à ChatGPT

## 🔐 Bonnes pratiques

- Clé API stockée dans les Secrets Colab (jamais dans le code)
- Code reproductible par n'importe quel utilisateur avec sa propre clé

## 💼 Applications business

Ces techniques permettent de construire :
- **Chatbots personnalisés** pour sites web et support client
- **Générateurs de contenu** (emails, articles, posts réseaux sociaux)
- **Assistants IA internes** (résumés, traductions, rédaction)
- **Automatisations marketing** (campagnes, newsletters)
